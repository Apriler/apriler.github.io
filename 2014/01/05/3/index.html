拥抱
发布于 2014-01-05 / 更新于 2019-01-13 / / 热度 0 / 喜欢 0

路灯收留了谁的寂寞，

也有我的一份，

不想被时间带走，

我不知道哪个是你，

所以我的梦被随意偷窃，

那是过去，

我的回忆，

碎了的天空透支了黑色的权利，

谁能杀死我，

或者给我一个拥抱，

可以不再梦。
